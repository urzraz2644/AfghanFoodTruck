# Afghan Food Truck Web Application
**ECPI SDC310L – Week 5 Final Submission** 
Author: Urzalla Razmowar • Date: November 9, 2025

## Overview
PHP + MySQL app that displays menu items for an Afghan food truck and includes Cart and Checkout pages.

## Features
- MySQL database: `afghan_food_truck`
- Pages: `menu.php`, `cart.php`, `checkout.php`
- Folders: `app/`, `public/`, `css/`, `images/`, `js/`, `sql/`
- Tested on XAMPP (Apache + MySQL), no errors

## Testing Summary
Menu, Cart, and Checkout pages loaded successfully. 
Database verified in phpMyAdmin (6 menu items). 
See Word doc in Canvas for screenshots and results.

## How to Run
1. Start Apache & MySQL in XAMPP 
2. Import `/sql/` export if needed 
3. Visit `http://localhost/AfghanFoodTruck/public/'
